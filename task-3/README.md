# Task 3: To-Do List Application

## Objective

Build an interactive to-do list manager that allows users to add, remove, and view items through a menu-driven interface.

## Setup Instructions

1. Navigate to the `task-3` directory
2. Run `dotnet run` to start the application

## What It Does

- Displays an interactive menu with four options:
  1. **Add Item** - Adds a new item to the list
  2. **Remove Item** - Removes an item from the list
  3. **Display Items** - Shows all items currently in the list
  4. **Quit** - Exits the application
- Continues running until the user selects the quit option

## Dependencies

- .NET 10.0 or later

## Example Usage

```
================================================
Pick an action :
1. add item
2. remove item
3. display items
4. quit
Enter your Input : 1
Enter your item to be added : Buy milk
```
