# Task 4: Student Grade Sorter with LINQ

## Objective

Explore LINQ (Language Integrated Query) by sorting and filtering a collection of students based on their academic grades.

## Setup Instructions

1. Navigate to the `task-4` directory
2. Run `dotnet run` to start the application

## What It Does

- Creates a collection of student records with name, grade, and age properties
- Uses LINQ query syntax to filter students (only those with grade > 0)
- Sorts students by grade in descending order
- Displays the sorted results in a formatted list

## Dependencies

- .NET 10.0 or later

## Example Output

```
[name:Ram and grade:9.2, name:Alex and grade:7.2, name:Ramesh and grade:4.2, name:George and grade:3.2]
```

## Key Concepts

- LINQ query syntax (from, where, orderby, select)
- Record types with properties
- IEnumerable collections
- Sorting and filtering with LINQ
- String interpolation and formatting
- Method overriding (`ToString()`)
