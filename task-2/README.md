# Task 2: Person Introduction

## Objective

Demonstrate the use of C# record types and basic object-oriented programming with methods.

## Setup Instructions

1. Navigate to the `task-2` directory
2. Run `dotnet run` to start the application

## What It Does

- Defines a `Person` record with name and age properties
- Creates multiple person instances
- Calls an `Introduce()` method on each person to display their information

## Dependencies

- .NET 10.0 or later

## Example Output

```
Hi, I am Ram and I am 25 years old
Hi, I am Alice and I am 20 years old
```

## Key Concepts

- C# record types for immutable data structures
- Class and record definitions
- Instance methods and object instantiation
- String interpolation
